# 步骤 3：生成依赖文件

根据选择生成依赖管理文件。

**FastAPI 项目通常包含的依赖**：
- fastapi - Web 框架
- uvicorn[standard] - ASGI 服务器
- sqlalchemy - ORM
- alembic - 数据库迁移工具
- pydantic - 数据验证
- pydantic-settings - 配置管理
- python-jose[cryptography] - JWT
- passlib[bcrypt] - 密码哈希
- python-multipart - 表单数据
- pytest - 测试框架
- httpx - 测试客户端

完成后展示文件内容并暂停等待确认。

---

# 步骤 4：创建 src/core/config.py

使用 question 工具确认需要的配置模块（可多选）。

然后生成 `src/core/config.py`，使用 Pydantic Settings 管理配置。

完成后展示代码并暂停等待确认。

---

# 步骤 5：创建 src/core/database.py

使用 question 工具确认数据库选项。

然后生成 `src/core/database.py`，包含：
- 数据库连接初始化函数（get_db）
- SQLAlchemy 引擎配置
- Session 工厂
- 生命周期管理函数（用于 FastAPI startup/shutdown）

完成后展示代码并暂停等待确认。

---

# 步骤 6：创建 src/core/security.py（可选）

使用 question 工具确认是否需要安全模块。

如果确认需要，生成 `src/core/security.py`，包含：
- password_hash.verify() - 验证密码
- password_hash.hash() - 哈希密码
- jwt.create() - 创建 JWT token
- jwt.decode() - 解析 JWT token

完成后展示代码并暂停等待确认。

---

# 步骤 7：创建 main.py

使用 question 工具确认 main.py 的配置。

然后生成 `main.py`，包含：
- FastAPI app 实例
- 生命周期事件（startup 初始化数据库，shutdown 关闭数据库）
- CORS 中间件
- 全局异常处理器
- 健康检查端点（GET /）
- 路由注册（占位，后续 S8 阶段填充）

完成后展示代码并暂停等待确认。
