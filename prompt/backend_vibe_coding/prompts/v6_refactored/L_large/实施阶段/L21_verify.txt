# 步骤 9：生成部署文档

创建 `docs/DEPLOYMENT.md`：

```markdown
# [项目名称] 部署文档

**版本**: v1.0

---

## 1. 环境要求

- Python: 3.11+
- 数据库: PostgreSQL 14+
- Redis: 7.0+

---

## 2. 安装步骤

### 2.1 安装依赖

```bash
pip install -r requirements.txt
```

### 2.2 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件
```

### 2.3 初始化数据库

```bash
alembic upgrade head
```

### 2.4 启动服务

```bash
uvicorn src.main:app --host 0.0.0.0 --port 8000
```

---

## 3. Docker 部署

### 3.1 构建镜像

```bash
docker build -t {项目名}:latest .
```

### 3.2 运行容器

```bash
docker-compose up -d
```

---

## 4. 健康检查

```bash
curl http://localhost:8000/health
```

---

## 5. 回滚方案

```bash
git checkout {previous_tag}
docker-compose up -d --build
```
```
